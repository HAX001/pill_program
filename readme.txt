注意：在wrape_***_pills.py文件中,从bmp到jpg的过程需要提前注意裁剪的大小！！！

1,有新图像数据，则请先存放于/data/origin_data/***_pills_1或***_pills_2/
2,剪切部分数据到相应的/data/origin_data/***_pills/；并对其进行筛选！！！（此处很关键！！！）筛选依据：没有目标的图像，
或者目标都在边缘，不方便标注的图像，这些图像都需要删除！
3,然后按以下步骤进行：
    1：运行1_bmp_to_jpg.py（注：该步骤不仅完成从bmp到jpg转化，同时完成排序工作，方便标注，还有
        如果开始数据完成裁剪了，则修改代码不用再裁剪，若是未裁剪，则修改代码，找到合适的值，裁剪之！）
    2：对/data/***_pills/图像进行标注(若2_标注可以使用的话，就用它吧，不行的话，请在网上找一下，有在win7下用opencv做的,
        我的这个就是打算仿它来写，写好就可以用，写不好就呜呜~~~~(>_<)~~~~了！)
    3：运行3_Annotation.py
    4：运行4_Imageset.py

按阅读顺序进行操作后，则可得到质量较好的数据集！注：***表示可选(white, blue, oill, yellow)
